## 기능목록
  - ### 일정관리
    - #### 일정추가
      - 사용자는 일정을 기입하면 데이터베이스에 저장된다.
    - #### 일정삭제
      - 사용자는 일정을 삭제하면 데이터베이스에서 제거된다.
    - #### 일정검색
      - 검색창에 일정제목 기입하면 검색내용과 일정제목이 연관된 일정 띄움
  - ### 자격증 AI
    - KNN 알고리즘 직무추천
      - 사용자에게 설정된 정보에 맞는 데이터와 크롤링으로 받아온 데이터를 모델링하여 추천 알고리즘 값 리턴
